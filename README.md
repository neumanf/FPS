# OpenGL Farm Scene

## Build

```sh
# Compila
$ g++ src/main.cpp -lm -lGL -lGLU -lglut -lSOIL -o farm
# Roda
$ ./farm
```

## Controles

| Atalho | Ação                             |
| ------ | -------------------------------- |
| W      | Movimenta a câmera para frente   |
| A      | Movimenta a câmera para esquerda |
| S      | Movimenta a câmera para trás     |
| D      | Movimenta a câmera para direita  |
| ESC    | Fecha a aplicação                |
